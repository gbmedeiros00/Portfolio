Projeto Acadêmico | Desempenho de Vereadores

O desafio proposto aos alunos é criar uma plataforma web que disponibilize informações sobre o desempenho dos vereadores da cidade durante o atual mandato. As informações estão disponibilizadas em https://camarasempapel.camarasjc.sp.gov.br/ O objetivo é oferecer aos eleitores dados claros e acessíveis que ajudem a tomar decisões informadas nas eleições municipais que se aproximam.
