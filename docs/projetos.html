PROJETOS

Projeto Acadêmico | Desempenho de Vereadores

O desafio proposto é criar uma plataforma web que disponibilize informações sobre o desempenho dos vereadores da cidade durante o atual mandato. As informações estão disponibilizadas em https://camarasempapel.camarasjc.sp.gov.br/ O objetivo é oferecer aos eleitores dados claros e acessíveis que ajudem a tomar decisões informadas nas eleições municipais que se aproximam

Projeto Acadêmico | Dashboard de Indicadores

O objetivo do projeto é desenvolver um Dashboard de Indicadores para monitorar e visualizar o impacto da plataforma ao longo do tempo. O painel apresentará métricas essenciais, como: número de lojas criadas, quantidade de usuários impactados e a distribuição geográfica das cidades atendidas. Com esses dados estruturados, será possível acompanhar o crescimento da iniciativa e fornecer informações valiosas para patrocinadores e stakeholders, garantindo maior transparência e embasamento estratégico para futuras decisões.
